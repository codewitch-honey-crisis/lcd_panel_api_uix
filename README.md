# lcd_panel_api_uix

A demonstration of using UIX with GFX and the ESP LCD Panel API

Supported devices:
- Lilygo TTGO T1 Display
- Lilygo T-Display S3
- Makerfabs ESP Display S3 Parallel w/ Touch
- Makerfabs ESP Display S3 4 inch w/ Touch
- Espressif ESP_WROVER_KIT 4.1
- M5Stack Core2
- M5Stack Fire
